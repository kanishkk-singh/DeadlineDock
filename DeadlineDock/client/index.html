Our story

About DeadlineDock

Making student life easier, one deadline at a time.

🎯
Our Mission
DeadlineDock helps students stay on top of every academic task — assignments, projects, exams. Built as a full-stack app: Express backend, JWT auth, JSON database, and a production-grade front end.
⚡
Tech Stack
Backend: Node.js + Express, bcrypt, JWT, helmet, compression, rate-limiting. Frontend: Vanilla JS ES6+, Chart.js, FullCalendar, custom cursor & particle canvas. No framework — pure performance.
👨‍💻
Meet the Creator
Built by a passionate CSE student who turned a simple localStorage app into a production-ready full-stack project. Every feature is real — real API, real database, real auth.
🚀
What's Next
PostgreSQL migration, team sharing, push notifications via Service Workers, AI-powered study plan generation, and a React Native mobile app. DeadlineDock is just getting started.
